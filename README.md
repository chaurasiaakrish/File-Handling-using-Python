# File-Handling-using-Python
This repository contains my hands-on practice and conceptual understanding of File Handling in Python. It covers reading, writing, appending, and working with different file modes.

## Topics Covered
- Opening and closing files
- File modes (r, w, a, rb, wb)
- Reading files:
- read()
- readline()
- readlines()
- Writing and appending data
- Using with statement (context manager)
- Working with text files
- Basic exception handling in file operations

## Concepts Practiced
- Creating files programmatically
- Updating file contents
- Handling file pointers
- Error handling (FileNotFoundError)
- Writing structured data into files
